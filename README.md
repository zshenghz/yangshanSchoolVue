# yangshanSchoolVue
在线教育网站后端

## 简介
* 后端采用SpringBoot+SpringCloud进行微服务架构，使用Feign、Gateway、Hystrix，以及阿里巴巴的Nacos等组件搭建了项目的基础环境。
* 前端部分使用主流的前端框架Vue，使用Es6的开发规范，采用模块化的开发模式。

照着尚硅谷的在线教育视频撸的。视频中的需求基本完成，权限管理也完成了。如果对你有帮助，点个☆star哦！！
